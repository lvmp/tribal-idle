# Art Bible: Tribal Idle

## Estilo Core
- **Estilo:** Pixel Art Moderno (estilo *Octopath Traveler* ou *Stardew Valley* de alta resolução).
- **Perspectiva:** Isométrica 3/4 (ângulo de 30 graus).
- **Resolução de Base:** 64x64 pixels para personagens, 128x128 para construções base.

## Paleta de Cores (Design Tokens)
- **Terra/Base:** `#8B4513`, `#A0522D`, `#D2B48C`
- **Fogo/Alerta:** `#FF4500`, `#FFD700`, `#FFFFFF` (brilho)
- **Natureza:** `#228B22`, `#006400`, `#F0E68C` (seco)

## Regra de Ouro da IA
"Todos os assets devem ter fundo branco puro (`#FFFFFF`) para facilitar a remoção automática (color keying) no Flame."
